# ビルド方法
--------------------

Makefile を使います。  

> すべてのソースをビルドする場合
$ make

> 指定ソースのみをビルドする場合 (.c を除いたファイル名を指定)
$ make 01_connect


# 必要なファイル
-----------------------

wayland-scanner で生成したヘッダファイルとソースファイルが必要な場合があります。
